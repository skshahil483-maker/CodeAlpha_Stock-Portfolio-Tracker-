# 📊 Stock Portfolio Tracker

A simple Python-based Stock Portfolio Tracker developed as part of the CodeAlpha Internship.

## 🚀 Features

- Add multiple stocks and their quantities
- Uses predefined stock prices
- Calculates total investment value
- Displays a portfolio summary
- Saves portfolio details to a text file
- User-friendly command-line interface

## 🛠️ Technologies Used

- Python
- Dictionaries
- Loops
- Conditional Statements
- File Handling
